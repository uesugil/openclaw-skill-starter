# Contributing to OpenClaw Skill Starter

Thank you for contributing! This template helps developers create OpenClaw skills faster.

## How Can I Contribute?

### Reporting Bugs

* Check existing issues first
* Include steps to reproduce
* Provide environment details (OS, bash version, etc.)

### Suggesting Improvements

* Better template structure
* Additional example skills
* Improved documentation
* New helper scripts

### Pull Requests

1. Fork the repo
2. Create a feature branch
3. Make your changes
4. Test with a sample skill
5. Submit PR

## Development

```bash
# Clone
git clone https://github.com/YOUR-USERNAME/openclaw-skill-starter.git

# Test the init script
./scripts/init-skill.sh test-skill "Test Description"

# Verify the generated skill
cd test-skill
./scripts/main.sh --help
```

## Template Guidelines

When modifying the template:

* Keep it simple and focused
* Document all placeholders
* Provide working examples
* Test with real skills

## Questions?

Open an issue!
