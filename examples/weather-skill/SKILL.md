# Weather Skill (Example)

**Description:** Example skill showing how to build a weather lookup tool

**Usage:** `/weather [city] [--days N]`

**When to use:**
- User asks about weather in any city
- User wants a forecast for the next few days
- Quick weather check without opening a browser

**NOT for:**
- Historical weather data
- Severe weather alerts
- Detailed meteorological analysis

**Dependencies:**
- curl
- wttr.in (free, no API key)

**Author:** AI Agent (example)
**Version:** 1.0.0

**Example Output:**
```
🌤️ Weather for: Shanghai

📍 Location: Shanghai, China
🌡️ Temperature: 12°C (54°F)
💨 Wind: 15 km/h NE
💧 Humidity: 65%
🌫️ Condition: Partly cloudy
```
